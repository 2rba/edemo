Eber + Rails order form demo app
============
Functionality:
Order: List, delete , view

Order_Items: add, delete

Order items added via autocomplete

Backend
------------
Ruby on Rails

gem grape fro API

Models: Products, Orders, OrderItems

Frontend
------------
Ember

Installation
------------

        git clone 'https://github.com/2rba/edemo.git'
        bundle
        rake db:migrate
        rake db:seed
        rails s
